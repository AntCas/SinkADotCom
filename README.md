SinkADotCom
===========

Sink A Dot Com from HeadFirst Java
Battleship inspired game from chs 5 & 6 of HeadFirst Jave [http://www.amazon.com/Head-First-Java-2nd-Edition/dp/0596009208]
3 defunct DotComs are located on a 7X7 grid, guess where they are to send them to a watery grave in cyber heaven.

To play: download the respository
Enter: java DotComBust into the console while in the repository
[NOTE] Must have JDK installed

Typical Game Play:

Your goal is to sink three dot coms.
Pets.com, eToys.com, Go2.com
Try to sink them all in the fewest number of guesses!
